Why would you use something like the load event? Does this event have disadvantages? Do you know any alternatives, and why would you use those?
You would use it to make sure that an image or script etc has completely loaded before executing certain actions. Helps prevent errors.
A disadvantage to the load event is that it doesn't work well with asynchronous code
An alternative could be something like DOMContentLoaded Event or Promises


What are the advantages and disadvantages of using Ajax?
Advantages:
Can update content without requiring a full page reload.
Smoother more responsive user experience
reduced bandwith


Disadvantages:
Adds complexity
Can pose security risks
Difficult to make more accessible to those with disabilities
